## 👨‍💻 About Me

Hey there! I'm **Siva Prasanna S**, a passionate **Full-Stack Developer** based in **Coimbatore, Tamil Nadu, India 🇮🇳**, who loves architecting end-to-end solutions — from pixel-perfect frontends to robust, scalable backends and cloud infrastructure.

- 🔭 Currently building **production-ready full-stack applications** using React, Spring Boot & Microservices
- 🌩️ Exploring **AWS Cloud** — working towards the **AWS Certified Cloud Practitioner (CLF-C02)** certification
- 🧩 Passionate about **clean architecture**, **RBAC**, **JWT security**, and **event-driven microservices**
- 🗄️ Comfortable with **relational & NoSQL databases** — MySQL, PostgreSQL, MongoDB
- 📖 Always learning — currently deep-diving into **Spring Cloud**, **Kafka**, and **Kubernetes**
- ⚡ Fun fact: I debug with coffee ☕ and deploy with confidence 🚀

- **Features:** Role-based access (Admin / Dev / Viewer), Sprint CRUD, Kanban board, React Hook Form + Yup validation

### 📝 Blog / CMS Application
